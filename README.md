# CineScout

## Live Link - https://cinescout.netlify.app/

Discover your next binge-worthy show or must-see movie.

## INSTALLATION

```
1. git clone https://github.com/TusharAnekar/cine_scout.git
2. cd cine_scout
3. npm install
4. npm start
```
